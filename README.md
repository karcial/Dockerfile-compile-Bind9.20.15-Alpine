# Dockerfile-compile-Bind9.20.15-Alpine
Image Docker personnalisée de BIND 9.20.15, basée sur Alpine Linux, compilée depuis les sources pour un serveur DNS optimisé, léger et sécurisé, avec prise en charge de DNSSEC, dnstap, LMDB et MaxMindDB, déployable facilement via Docker avec volumes persistants et exposition du port 53 TCP/UDP.
